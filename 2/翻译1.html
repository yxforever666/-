非线性微分方程系统描述

假设，控制对象由以下非线性微分方程系统描述：

$$\dot{x}(t) = f(x, u),$$

其中 \(x\) 是状态向量，维度为 \(n\)；\(u\) 是控制向量，维度为 \(m \leq n\)。 提出第一个问题，即协同控制理论的最简单问题：需要找到控制规律

$$u(\psi) = u(x),$$

它能够将系统(1.10)的成像点（IT）从某个初始状态 $$x_0(x_{10}, \ldots, x_{n0})$$ （在某个允许的区域内）首先转移到状态空间中的不变流形

$$\psi(x_1, \ldots, x_n) = 0$$

的邻域中，然后沿着该流形的轨迹渐近稳定地移动到系统(1.10)的期望状态，尤其是移动到坐标的起点。 控制规律 (1.11) 在后续沿着 $$\psi(x_1, \ldots, x_n) = 0$$ (1.12) 的运动中，保持 IT 位于指定邻域内。

不变流形 $$\psi(x_1, \ldots, x_n) = 0$$ 可以解释为指定的目标集合，IT 必然从任意初始状态吸引到这个集合，然后沿着它运动。 例如，对于二阶对象，流形表示所需的相位轨迹 $$\psi(x_1, x_2) = 0$$， IT 会先接近它，然后沿着相位平面原点的轨迹运动。

主控流形 $$\psi(x_1, \ldots, x_n)$$ 应满足如下的泛函方程：

$$T \psi(t) + \varphi(\psi) = 0,$$

其中当 $$\varphi(\psi) > 0$$ 且 $$T > 0$$ 时，它是关于稳定极值的欧拉-拉格朗日方程，提供伴随优化功能 $$J_x$$ 的最小值：

$$J_x = \int_0^\infty \left[\varphi^2(\psi) + T^2 \dot{\psi}^2(t)\right] dt,$$

这是闭合系统运动轨迹上的功能。
