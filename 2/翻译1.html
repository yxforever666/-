非线性微分方程系统描述

假设，控制对象由以下非线性微分方程系统描述：

𝑥̇(t) = 𝑓(x, u),

其中 x 是状态向量，维度为 n；u 是控制向量，维度为 m ≤ n。 提出第一个问题，即协同控制理论的最简单问题：需要找到控制规律

u(ψ) = u(x),

它能够将系统(1.10)的成像点（IT）从某个初始状态 x₀(x₁₀, …, xₙ₀)（在某个允许的区域内） 首先转移到状态空间中的不变流形

ψ(x₁, …, xₙ) = 0

的邻域中，然后沿着该流形的轨迹渐近稳定地移动到系统(1.10)的期望状态，尤其是移动到坐标的起点。 控制规律 (1.11) 在后续沿着 ψ(x₁, …, xₙ) = 0 (1.12) 的运动中，保持IT位于指定邻域内。

不变流形 ψ(x₁, …, xₙ) = 0 可以解释为指定的目标集合，IT必然从任意初始状态吸引到这个集合，然后沿着它运动。 例如，对于二阶对象，流形表示所需的相位轨迹 ψ(x₁, x₂) = 0，IT会先接近它，然后沿着相位平面原点的轨迹运动。

主控流形 ψ(x₁, …, xₙ) 应满足如下的泛函方程：

Tψ(t) + φ(ψ) = 0,

其中当 φ(ψ) > 0 且 T > 0 时，它是关于稳定极值的欧拉-拉格朗日方程，提供伴随优化功能 Jₓ 的最小值：

Jₓ = ∫₀^∞ [φ²(ψ) + T²ψ̇²(t)] dt,

这是闭合系统运动轨迹上的功能。
